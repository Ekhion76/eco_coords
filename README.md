# ECO COORDS
- Save coords in server side
- Copy to clipboard

## COMMANDS:
'coords' ON / OFF
'resetcoords' set default settings 

## CONTROLS

SCROLL or NUMPAD + / -

CHANGE MARKER Z POSITION

SHIFT + SCROLL or NUMPAD + / -

CHANGE MARKER SIZE

SHIFT + CTRL + SCROLL or NUMPAD + / -

CHANGE MARKER TYPE
