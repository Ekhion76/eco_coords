# ECO COORDS
- Save coords server side in txt file
- Copy to clipboard

## COMMANDS:
'/coords' ON / OFF
'/resetcoords' set default settings 

## CONTROLS

SCROLL or NUMPAD + / -

change marker z position

SHIFT + SCROLL or NUMPAD + / -

change marker size

SHIFT + CTRL + SCROLL or NUMPAD + / -

change marker type
